Zip folder included:
 - Docs folder: Innehåller Manual vilket är en pdf med instruktioner och material angående de två programmen och projektet, teknisk dokumentation i pdf form och en README fil som vägleder i projektets filstruktur.
 - Acceptanstester folder: Innehåller färdiga acceptanstester av alla stories för användning under release, alla kategoriserade för olika stories.
 - register.jar: Är register programmet som kan användas för att registrera start/måltider för att sedan sparas på fil.
 - result.jar: Är result programmet som kan användas för att generera resultatet tillsammans med namnfil,starttider och måltider.
 - release-info: Beskriver alla stories och filer som levereras i nuvarande release.
 - configfile.json: Är Konfigurationsfilen som används för result.jar programmet, alla ändringar för konfigurationer görs där.
